Welcome! This is the official Helm chart repository for the DevOps1910-Project.

How to use this repository:

1. Add the repository to Helm:

helm repo add my-flask-repo https://edenshark24.github.io/DevOps1910-Project/Helm/helm-repo/

2. Update your local cache:

helm repo update

3. Search for available charts:

helm search repo my-flask-repo
